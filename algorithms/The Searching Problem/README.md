The Searching Problem
=====================

####Input####
Sequence of n numbers A = [a1, a2, …, an] and a value V
####Output####
An index i such that V = A[i] or NIL if V doesn’t exist in A
####Example####
* A=[1,2,3,4,5,4]
* V=3
* f(A, V) = 2 (considering 0 as the first index)

* TCRC 3rd Edition - Problem 2.1-3
