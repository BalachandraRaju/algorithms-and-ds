The Sorting Problem
===================

####Input####
Sequence of `N` numbers `A = [a1, a2, a3, ..., aN]`

####Output####
Permutation of the `N` elements in `A` such that `[a1 <= a2 <= a3 <= ... <= aN]`

####Example####
1. `A=[9,8,7,6,5,4,3,2]`
2. `f(A)`
3. `A=[2,3,4,5,6,7,8,9]`
